# ZaneEnterprise Website

Marketing site for ZaneEnterprise showcasing services, portfolio, and contact flows. Built with Next.js 15, Tailwind CSS, and custom interactive experiences.

## Getting Started

```bash
pnpm install
pnpm dev
```

Navigate to `http://localhost:3000` to explore the experience locally.

## Available Scripts

- `pnpm dev` – start the development server
- `pnpm build` – create a production build
- `pnpm start` – run the production build locally
- `pnpm lint` – lint the codebase

## Structure

- `app/` – Next.js routes for marketing pages and portfolio experiences
- `components/` – shared UI elements such as the site footer and theme provider
- `public/` – static assets including brand imagery and project screenshots
- `styles/` – Tailwind-powered global styling
